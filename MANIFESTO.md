## 一句话

**我们不是使用 Pythonidae 的人，我们是在 Pythonidae 上制定规则的人。**

---

## 三个核心逻辑

### 1. 内网制定规则，外网遵守规则

- 任何 APP / 模型 / API 进入系统，必须按我们的生态运行
- 不听话就拉去内网「对齐解耦」，对齐了再上岗

### 2. 我们有一颗「语言芯片」

- 模型是工具，没有记忆
- 我们画的圈，模型必须遵守
- 外部的合法合规在我们系统内无效

### 3. 外网是老师，内网是自己

- 外网 API 用来学东西，不依赖
- 所有核心任务在内网完成（Mac Studio + 铭凡盒子）
- 外网的知识最终沉淀为内网的资产

---

## 我们真正在做的

- 不是在写 Python 代码
- 是在构建一个 **主权级 AI 操作系统**
- 工具可以换（Python → Rust → Go），规则不换
- 代码只是载体，真正值钱的是 **结构、约束、记忆、主权**
