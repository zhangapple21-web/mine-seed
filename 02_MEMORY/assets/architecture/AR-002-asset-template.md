# AR-002 Asset Template（资产模板）

## Name
AR-002 — Asset Template

## Origin
AUM-MISSION-ARCH-017 文明层设计。

## Purpose
所有 Civilization Asset 必填的 10 个字段。

## Problem
无模板的资产难以被后续 Agent 理解与索引。

## Core Structure
```markdown
# XX-NNN Asset Name

## Name
## Origin
## Purpose
## Problem
## Core Structure
## Constraint
## Evidence
## Distillation
## Related Assets
## Replaceable
## Rebuildable
```

## Constraint
- 11 字段缺一不可
- Distillation 必须为单段直白结论
- Related Assets 必须交叉引用

## Evidence
- 22+ 资产已使用该模板（2026-07-14）

## Distillation
「模板是文明的语法」。

## Related Assets
- GV-001 Admission Engine
- PR-003 Distill Before Archive
- AR-001 Four-Layer Architecture

## Replaceable
可替换（字段可加可减）。

## Rebuildable
可重建。读「10 字段模板」即可重建。
