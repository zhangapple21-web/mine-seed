# CG-002 Environmental Awareness（环境感知）

## Name
CG-002 — Environmental Awareness

## Origin
用户偏好「环境感知优先于自动学习」。

## Purpose
系统应先感知环境再行动，不是盲目自动学习。

## Problem
盲目自动学习导致在错误环境中浪费时间。

## Core Structure
持续运行环：
```
Perception → Hypothesis → Debate → Decision → Execution → Observation → Experience
```

Perception 永远先于 Action。

## Constraint
- 任何执行前必须有 Perception
- Perception 必须基于 Environment First Protocol (EFP)
- 不得跳过感知直接执行

## Evidence
- EFP（Environment First Protocol）启动流程
- Recovery Protocol (RP) 触发前先扫描环境

## Distillation
「环境先行，行动在后」。

## Related Assets
- PT-001 Heartbeat Protocol
- PR-001 Drawer First
- EFP-001 Environment First Protocol

## Replaceable
可替换（感知粒度可调）。

## Rebuildable
可重建。读「环境先于行动」即可重建。
