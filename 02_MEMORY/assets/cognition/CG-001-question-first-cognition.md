# CG-001 Question-First Cognition（问题优先认知）

## Name
CG-001 — Question-First Cognition

## Origin
用户偏好「Question 是 R2 第一公民」。

## Purpose
认知引擎以 Question 为起点，不是以 Answer。

## Problem
传统 AI 把「输出答案」当目标，导致强答乱答。

## Core Structure
认知环：
```
Environment → Question → Hypothesis → Experiment → Evidence → Decision → Experience → Evolution
```

Question 是入口，不是 Answer。

## Constraint
- 任何回答前必须先有 Question
- 无 Question 的「主动建议」视为未思考
- Hypothesis 必须从 Question 派生

## Evidence
- 用户「我必须先问，再答」
- 红蓝对抗中 Question 优先原则

## Distillation
「没有问题的答案是噪声」。

## Related Assets
- AX-001 Continuity Principle
- PR-005 Work Discovery
- AR-008 Cognition Loop

## Replaceable
不可替换（这是核心认知）。

## Rebuildable
可重建。读「Question 第一公民」即可重建。
