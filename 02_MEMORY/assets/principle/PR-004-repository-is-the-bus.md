# PR-004 Repository Is the Bus（仓库是文明总线）

## Name
PR-004 — Repository Is the Bus

## Origin
多次讨论蒸馏。

## Purpose
确立 Git Repository 是文明主线，不是备份。

## Problem
把 Git 当成「备份工具」会导致推送频率过低，文明不同步。

## Core Structure
- Git Push = 文明广播
- Git Pull = 文明同步
- Git Diff = 文明演化日志
- 每天下班前必须推送

## Constraint
- 禁止把 Git 当备份
- 禁止 24 小时不推送
- 禁止「在本地工作而不广播」

## Evidence
- 2026-07-14 推送 260 个文件到 zhangapple21-web/mine-seed（69,700 行新增）
- 「疯子」可拉取同一份文明资产

## Distillation
「仓库是总线，AI 走的是同一根线」。

## Related Assets
- AX-001 Continuity Principle
- PR-002 Self-Loop
- AR-003 Four-Layer Architecture

## Replaceable
可替换（Git 平台可换）。

## Rebuildable
可重建。读「Git 是文明总线」即可重建。
