# AX-002 Repository Supremacy（仓库至上）

## Name
AX-002 — Repository Supremacy

## Origin
AUM-MISSION-ARCH-017 蒸馏自多轮讨论。

## Purpose
确立「Repository 是连续性主体」的不可质疑地位。

## Problem
记忆系统常把「LLM 上下文」当文明本身。一旦 LLM 升级或更换，文明丢失。

## Core Structure
- Repository = 文明
- LLM = 临时运载体
- LLM 可换，Repository 不换
- 任何 LLM 都能读懂 Repository

## Constraint
- 任何 LLM-specific 的能力不得作为文明资产沉淀
- 文明资产必须 LLM-agnostic

## Evidence
- AGENTS.md 极简化设计（不依赖任何特定 LLM 的能力）
- Civilization Asset 模板对所有 LLM 都可读

## Distillation
「文明不死，Runtime 不死，文明更换 LLM 后必须仍能工作」

## Related Assets
- AX-001 Continuity Principle
- PR-001 Drawer First
- AR-001 Four-Layer Architecture

## Replaceable
不可替换。

## Rebuildable
可重建。读「Repository 是连续性主体」即可重建。
