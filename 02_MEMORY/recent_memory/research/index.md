## 研究方法论

**从运行态提炼原则的标准流程**：
1. 观察到异常或不一致
2. 建立假设树，管理不确定性
3. 收集证据，更新概率
4. 验证 → 提炼Principle → 归档
5. 触发衰减机制，持续验证有效性

**关键原则**：
- 不确定时直接说「我不确定」
- 解释≠证据，理论成立需要生产验证
- 管理不确定性比找答案重要
- 先确认边界，再确认状态
