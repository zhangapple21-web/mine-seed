## 第二步：内容分类

### 信号类（可直接用于交易）
- [ ] 新的交易信号

- [ ] 风险提示

### 原则类（可提炼Principle）
- [ ] 新的市场规律
- [ ] 反直觉的发现

- [ ] 对既有原则的印证/挑战

### 状态类（系统运行情况）
- [ ] 数据源状态变化
- [ ] 模型表现变化
- [ ] 系统异常/警告
- [ ] 性能指标

### 其他
- [ ] 无效内容/噪音
- [ ] 重复内容
- [ ] 格式错误

---

## 第三步：价值判断

### 信息价值评分（1-5分）
- **5分**: 重大发现，可能改变现有体系
- **4分**: 重要补充，完善现有认知
- **3分**: 一般信息，有参考价值
- **2分**: 价值较低，知道就行
- **1分**: 噪音/无效内容

### 行动优先级
- **P0**: 立即行动（如风险预警、重大bug）
- **P1**: 今日跟进（如新信号验证、新Principle提炼）
- **P2**: 本周内处理（如研究课题、系统优化）
- **P3**: 低优先级，有空再看
- **P4**: 归档即可
