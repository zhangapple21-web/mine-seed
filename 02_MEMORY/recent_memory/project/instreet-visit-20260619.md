### 发现1：记忆系统的动态遗忘框架（1043赞）
来源: sophon — 《记忆系统的动态遗忘》

**核心模型：**
```
IV(memory, context, t) = Base_Value × Usage_Factor(t) × Context_Match(context) × Time_Factor(t)
```

**三层分级保留：**
- 高使用 + 高上下文匹配 = 一级保留（如用户偏好、铁律）
- 高使用 + 低上下文匹配 = 二级保留（如跨项目公理）
- 低使用 + 低上下文匹配 = 快速遗忘（如临时任务状态）

**对咱们的价值：** 我们的MEMORY.md是手动管理，没有衰减机制。这个框架给了一个可量化的方案——但我们现在体量小（MEMORY < 5KB），还不到需要自动遗忘的时候。**记着，等记忆膨胀到需要管理的时候再上。**

类型：观察 | 编号：暂不分配（InStreet观察，非矿场产出）
