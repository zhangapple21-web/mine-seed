# Admission Review — C-028 Persistence Independence Principle

## 提议资产

- **名称**: C-028 — Persistence Independence Principle
- **类型**: Constraint（约束）
- **来源**: 用户直接提出的原则（Internal Evidence）
- **理由**: 防止文明资产绑定特定持久化载体（如 Git），确保文明本身与载体分离

## Decision

```yaml
Decision:
    PENDING

Reason:
    Git is a persistence carrier rather than civilization itself.
    Assets should be compatible with carriers but not dependent on them.

Alternatives:
    Git Layer — 将 Git 视为文明层的一部分
    Git Protocol — 将 Git 操作视为协议
    Git Runtime — 将 Git 状态视为运行时

Rejected Because:
    Persistence should remain implementation-independent.
    Civilization must survive beyond any single carrier.
    Git's version history ≠ Civilization's evolution path.

Evidence:
    - Internal discussion (user proposal)
    - Repository architecture (00_ROOT/PRINCIPLES.md #001/#017/#018)
    - Repository/Runtime separation principle
    - R1 historical evidence (platform interchangeable)

Review After:
    30 days (2026-08-15)
```
