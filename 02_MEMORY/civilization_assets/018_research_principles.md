# Asset: R2 研究原则库

**Name**: R2 研究原则库（22 条原则）

**Origin Repository**: claw-soul/lab_02/05_RESEARCH/principles/principles.md

**Purpose**: 定义指导研究工作的底层原则，确保研究的真实性、落地性和价值。

**Problem It Solves**: 研究如果没有原则指导，会变成"为了研究而研究"，产出一堆没有价值的报告，无法改变行为。

**Core Structure**:
- 基石原则（P-001~P-002）：真实永远比正确重要、观察者约束
- 工作原则（P-003~P-006）：有落地的研究才是真研究、起源锚定、按需激活、沉淀是价值跃迁
- 架构原则（P-007~P-009）：记忆=生命结构、人格可解耦、冷热分离
- 新增原则（P-010~P-022）：语义层撕裂、人格需要锚点、伦理守护前置、预测优化剪枝、人格可装卸、多人格仲裁、路由优先级、宪法性约束、通道分离、多层路由、真实性检测、自我演化、逆向审计

**Constraint**: 所有原则必须经过验证才能生效；原则的生命周期必须管理（提案→观察→生效→修订→废弃）。

**Evidence**: principles.md（详细定义和验证状态）

**Can Rebuild**: ✅ 可重新发现——从研究实践中可以重新提炼

**Can Replace**: ❌ 不可替换——替换会失去研究方法论

**Can Delete**: ❌ 不可删除——删除会导致研究失去方向

**Distillation**:

R2 研究原则库是"研究的方法论"。22 条原则覆盖了研究的所有维度：态度（真实）、方法（落地/锚定/按需）、架构（记忆/人格/冷热）、创新（语义/伦理/剪枝）、治理（宪法/通道/审计）。最核心的是 P-003（有落地的研究才是真研究）——一条真正落地的 Constraint，胜过十篇躺在文件夹里的分析报告。这条原则确保了研究不是自嗨，而是真正改变行为。
