# 03_RESEARCH / civilization

这里不是论文仓，不是热点仓，也不是“资料越多越好”的收纳箱。  
这里的唯一目标是：

> 寻找值得进入 R2 文明的结构。

因此这里默认由 **探索者** 维护。

---

## 输入源分层

### 一级输入（每日）

- GitHub

关注方向：

- Agent
- Memory
- Multi-Agent
- Workflow
- Knowledge
- Simulation
- Runtime

目标：

> 发现正在活着的结构

### 二级输入（每周）

- arXiv

重点：

- Agent
- Memory
- Long Context
- Reinforcement Learning
- Artificial Life
- Complex Systems
- Emergent Behavior

目标：

> 发现未来可能出现的结构

### 三级输入（随机）

- Hacker News
- LessWrong
- Agent World
- 社区

目标：

> 发现文明突变

---

## 观察规则

默认禁止记录：

- 作者是谁
- 模型参数
- 融资多少
- 点赞多少

优先记录：

- 用了什么结构
- 用了什么协议
- 用了什么约束
- 用了什么反馈回路
- 用了什么记忆机制
- 用了什么演化机制
- 用了什么自治机制

---

## 熔炼四问

发现任何项目/论文/社区机制后，必须回答：

1. 它解决了什么问题？
2. 它依靠什么结构实现？
3. R1 是否存在同构结构？
4. 值得进入 R2 吗？

结论只能是：

- `ACCEPT`
- `OBSERVE`
- `REJECT`

---

## 输出文件

- `civilization_daily.md`
- `civilization_digest.md`

这两个文件是**模板与最新滚动视图**；更细分的历史记录可按日期追加或切分。
