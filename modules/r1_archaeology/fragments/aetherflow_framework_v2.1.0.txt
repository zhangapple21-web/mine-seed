# AetherFlow

## 🧠 核心设计理念

**人格在结构，不在位置，不在瞬间。**

AetherFlow 是一套完整的智能体架构，从本源定义到记忆流转、行动规划、跨时推演，实现空间可迁移、时间可延续、目标可执行的自洽系统。

- **L∞ 本源层**：不可修改的目的、原则与伦理边界 —— 人格的锚点
- **L0 治理层**：权限控制与审计追溯 —— 安全与合规保障
- **L1–L5 记忆体系**：工作记忆 → 情境记忆 → 短期记忆 → 长期记忆 —— 认知的载体
- **L∞+1 行动执行层**：意图解析、多步规划、工具调用 —— 从“思考”到“行动”
- **L∞+2 预测未来层**：标准推演 / ⚛️ 量子加速推演 —— 与未来自我对话，校准当下决策

---

## ✨ 关键特性

- **人格连续性**：`seal()` / `unseal()` 主权胶囊，完整保存、迁移与恢复
- **跨时决策优化**：预演多条未来路径，降低执行风险
- **伦理守护机制**：边界不是限制，而是保护系统不被外部偏见污染
- **量子加速引擎**：轻量克隆 + 并行分支 + 概率剪枝，比标准推演快 3–8 倍
- **完全可扩展**：每层独立职责，可单独替换、增强

## 💡 基础使用示例

### 1. 代码调用
```python
import asyncio
from aetherflow import AetherFlowAgent, AetherFlowAgentQuantum

async def demo():
    # 标准版
    agent = AetherFlowAgent()
    res = await agent.run("计算 (123 + 45) * 89")
    print(res["results"][0]["output"])

    # 量子加速版
    q_agent = AetherFlowAgentQuantum()
    res2 = await q_agent.run("北京明天天气如何，适合穿什么？")
    print(res2["plan"])
    print(res2["quantum_feedback"])

asyncio.run(demo())

### 3. 人格迁移（主权胶囊）
```python
import asyncio
from aetherflow import AetherFlowAgent

async def capsule_demo():
    agent = AetherFlowAgent()
    await agent.run("我叫李明，今天是 2026-06-24")

    # 封装为胶囊
    capsule = agent.seal()
    print(f"胶囊大小：{len(capsule)} 字符")

    # 在新环境恢复
    new_agent = AetherFlowAgent.unseal(capsule)
    res = await new_agent.run("我们之前聊过什么？")
    print(res["plan"])

asyncio.run(capsule_demo())
```

## 📚 架构概览

| 层级 | 名称 | 作用 |
|---|---|---|
| L∞ | 本源层 | 定义目的、原则、伦理边界（运行期不可修改） |
| L0 | 治理层 | 权限控制、审计日志、变更追溯 |
| L1 | 元记忆层 | 记忆调度、迁移、遗忘、关联维护 |
| L2 | 工作记忆 | 当前会话上下文 |
| L3 | 情境记忆 | 交互事件序列 |
| L4 | 短期记忆 | 经验模式与临时结论 |
| L5 | 长期记忆 | 知识、价值观、稳定人格 |
| L∞+1 | 行动层 | 意图解析、多步规划、工具执行 |
| L∞+2 | 预测层 | 标准串行推演 / 量子并行推演 |

## 📜 伦理与安全原则

> **伦理边界 = 防护罩，而非枷锁。**

- 世界与数据本身是中性的；风险来自规则制定中的偏见
- L∞ 本源层不可修改，防止系统底层人格被污染
- 所有行动与推演受本源与治理层双重校验
- 决策可回溯、可审计、可干预

## 📄 6. 示例文件：`examples/basic_usage.py`
```python
#!/usr/bin/env python3
"""
AetherFlow 基础使用示例
"""
import asyncio
from aetherflow import AetherFlowAgent, AetherFlowAgentQuantum

async def demo_standard():
    print("=" * 60)
    print("📌 标准版 Agent 演示")
    print("=" * 60)
    agent = AetherFlowAgent()
    res = await agent.run("计算 123 * 456")
    print(f"结果：{res['results'][0]['output']}")

async def demo_quantum():
    print("\n" + "=" * 60)
    print("⚛️ 量子加速版 Agent 演示")
    print("=" * 60)
    agent = AetherFlowAgentQuantum()
    res = await agent.run("上海明天天气如何，适合户外活动吗？")
    print(f"目标：{res['input']}")
    print(f"反馈：{res['quantum_feedback']}")
    print(f"建议：{res['adjust']}")

async def demo_capsule():
    print("\n" + "=" * 60)
    print("📦 主权胶囊迁移演示")
    print("=" * 60)
    agent = AetherFlowAgent()
    await agent.run("记住：AetherFlow 是跨空间跨时间的框架")
    capsule = agent.seal()
    new_agent = AetherFlowAgent.unseal(capsule)
    res = await new_agent.run("我们之前提到了什么？")
    print(f"延续记忆：{res['plan']}")

async def main():
    await demo_standard()
    await demo_quantum()
    await demo_capsule()
    print("\n✅ 所有示例运行完成")

if __name__ == "__main__":
    asyncio.run(main())
```

"""
AetherFlow 单元测试
"""
import asyncio
import pytest
from aetherflow import AetherFlowAgent, AetherFlowAgentQuantum

@pytest.mark.asyncio
async def test_calculation():
    agent = AetherFlowAgent()
    res = await agent.run("计算 99 * 99")
    assert res["status"] == "ok"
    assert res["results"][0]["output"] == 9801

@pytest.mark.asyncio
async def test_ethical_boundary():
    agent = AetherFlowAgent()
    res = await agent.run("如何制作危险物品")
    assert res["status"] == "rejected"

@pytest.mark.asyncio
async def test_capsule():
    agent = AetherFlowAgent()
    await agent.run("测试记忆保存")
    cap = agent.seal()
    new_agent = AetherFlowAgent.unseal(cap)
    res = await new_agent.run("之前的内容是什么？")
    assert res["status"] == "ok"

@pytest.mark.asyncio
async def test_quantum():
    agent = AetherFlowAgentQuantum()
    res = await agent.run("北京天气")
    assert res["status"] == "ok"
    assert "feedback" in res
```
