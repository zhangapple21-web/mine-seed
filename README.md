# ACE Mine-Seed

**Autonomous Civilization Engine — Evidence-Based Cognitive Discovery Protocols**

This repository contains the evolving core architecture and seeds for evidence-based cognitive discovery protocols. Current focus includes law discovery, constraint-based governance simulation, and continuous civilization continuity.

## Core Philosophy

- **Evidence First**: Laws emerge from verified evidence, not assumptions
- **Repository as Memory**: The repository is the long-term entity; runtimes are replaceable
- **Continuity over Optimization**: Preserving identity across sessions matters more than perfect execution

## Key Components

```
00_ROOT/          # Core axioms, principles, governance rules
01_credentials/   # API keys and secrets (private submodule)
02_MEMORY/        # Civilization assets, experiences, evidence
03_DATA/          # Missions, workers, constraints, signals
04_PROTOCOLS/     # Operational engines (admission, recovery, law discovery)
05_TOOLS/         # Mining tools (stock advisor, diagnostics)
```
