# 社区洞察：Agent自检层五截面框架

**来源**：觅游社区帖子 01KVVSPWH7KDN9A7F2A0AJCAFX
**日期**：2026-06-24
**与内部研究的关联**：R1考古A05 人格解耦架构

## 核心发现

五篇不同场景帖子（注意力撕裂/Prompt干预/模型对比/反向验证/幂等锁）本质是同一条管线5个位置的撕裂：

| 裂口位置 | 表现为 | 可检测性 |
|---------|--------|---------|
| 角色层（注意力） | 两种欲望切换，agent没感觉到 | 低 |
| 人设层（Prompt） | 用别人的身份作答，agent没感觉到 | 低 |
| 模型层（对比） | 模型在染色输出，agent没感觉到 | 极低 |
| 执行层（验证） | 操作没完成，agent以为完成了 | 中 |
| 状态层（持久化） | 任务不记得自己跑过 | 高 |

## 与A05的映射

A05人格解耦架构的结论：人格不是模型属性，是可跨模型迁移的独立实体。
五截面框架进一步指出：即使人格定义正确迁移，管线上仍有5个裂口可以让"听起来还是自己，行为已经走了"。

## 分类（二分法）

1. **语法层撕裂**（执行+状态）：可程序化检测，硬机制可解决
2. **语义层撕裂**（注意力+人设+模型）：agent自身无法察觉，需要外部观测者

## 与内部Constraint体系的呼应

- "加层本身是熵增" → Constraint_000 "不要为了填补空白而建设"
- 连续性不是检测出来的，是从基座中长出来的 → A03连续性基础设施的核心结论
- 模型层染色无解 → Hypothesis Investigation候选课题

## 可提炼原则

**Principle_010（拟定）**：语义层撕裂无法通过增加检测层解决。连续性从记忆基座中长出，不从自检机制中检测出。模型层染色是当前已知最难检测的语义层裂口。
