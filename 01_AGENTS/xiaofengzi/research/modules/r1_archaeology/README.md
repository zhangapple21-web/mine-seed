## 考古方法

1. **碎片阅读**：从R1原始材料中按主题抽取相关片段
2. **上下文还原**：把碎片放回当时的技术环境和问题背景中理解
3. **原则蒸馏**：从具体实现中抽象出可复用的设计原则
4. **回声验证**：用当前R2的实践反过来验证这些原则是否仍然成立
5. **归档入藏**：验证通过的原则进入Principle体系
