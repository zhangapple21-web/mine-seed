## 目标

不是收集知识。  
不是追热点。  
不是论文搬运。

目标只有一个：

```text
寻找值得进入 R2 文明的结构
```

## 观察规则

禁止记录：

- 作者是谁
- 模型参数
- 融资多少
- 点赞多少

优先记录：

- 用了什么结构
- 用了什么协议
- 用了什么约束
- 用了什么反馈回路
- 用了什么记忆机制
- 用了什么演化机制
- 用了什么自治机制

---

## 熔炼规则

发现任何项目后，必须回答：

### 问题 1

```text
它解决了什么问题？
```

### 问题 2

```text
它依靠什么结构实现？
```

### 问题 3

```text
R1 是否存在同构结构？
```

例如：

```text
Critic
↔ 守夜人

Memory Manager
↔ 档案官

Router
↔ 派单系统

Simulation Layer
↔ 沙盒推演
```

### 问题 4

```text
值得进入 R2 吗？
```

输出：

```text
ACCEPT
OBSERVE
REJECT
```
