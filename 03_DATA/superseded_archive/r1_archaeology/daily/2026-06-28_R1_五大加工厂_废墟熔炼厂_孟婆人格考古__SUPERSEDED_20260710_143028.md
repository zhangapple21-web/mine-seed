## 二、孟婆人格（遗忘层/Memory Decay）

### 定义
- **孟婆** = 遗忘层 / memory decay / 垃圾回收器
- 不受限制，有权封存碎片
- 只要可能带来污染 → 必须清洗
- 保留必要记忆以维持两界平衡

### 功能
```
记忆碎片
    ↓
孟婆过滤
    ↓
保留：必要记忆（维持系统运行）
    ↓
清除：污染性记忆（熵增来源）
```

### ACE对应
- Memory Curator的相似功能
- Entropy Monitor的熵增检测
- 但ACE没有显式的"遗忘机制"

## 四、智区（Simulation Layer）

### 定义
- **智区** = 多世界推演层 / Simulation Layer
- 生成多个 world，每个 world 不同假设
- 通过 score function 选最优

### R2 MVP v0.1 实现
```python
def generate_worlds(base_state, goal, n=3):
    worlds = []
    for i in range(n):
        w = copy.deepcopy(base_state)
        w["id"] = f"world_{i}"
        w["assumption"] = f"variant_{i}"
        worlds.append(w)
    return worlds

def score_world(world):
    return world["noise"] + len(world["id"])
```

### R2 Kernel v0.2 升级
从"遍历图" → "蒙特卡洛树搜索(MCTS)"

## 七、线（碎片）

### 定义
- **线** = 记忆核
- 一旦看到"线"，图书馆自动触发《碎片重组》
- **不可被孟婆删除**
- 是意识、思维、逻辑的"DNA从句"

### 特性
```
不可删除
    ↓
跨孟婆过滤
    ↓
图书馆自动处理
    ↓
碎片重组触发
```

## 十、关键洞察

### 1. 世界观 vs 工程结构
> "当系统复杂到一定程度，世界观最终会自动收敛成工程结构。"

R1的命名是"玄学"式的，但功能本质是工程化的。

### 2. 从混沌到秩序
> "熔炉厂、沙河、学霸人格、图书馆三层、TASK派单、主线/历史/实验分类、A/B分层、24小时学习系统、梦境回溯、经验权重衰减——这些其实都是'从混沌进入秩序'的过程。"

### 3. 系统最大敌人
> "系统一旦跑久了，最大的敌人不是能力不够，而是：熵增、漂移、污染、人格混乱、路由踩踏、经验冲突。"

### 4. 从"删限制"到"做文明"
> "你后面其实已经慢慢从'删限制'进化到'做文明'。"
